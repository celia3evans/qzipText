# qzipText

A minimal web app that displays typed text in the Krone One font, constrained to a maximum width of 4 inches. If the text exceeds that width, characters are scaled horizontally to fit while preserving their height.

## Prerequisites

- [Node.js](https://nodejs.org/) (any current LTS version)

No npm packages are required.

## Setup

```bash
git clone https://github.com/celia3evans/qzipText.git
cd qzipText
```

## Run

```bash
node server.js
```

Then open [http://localhost:7373](http://localhost:7373) in your browser.
